## 🧠 香农网格交易法深度解读

> "通过智能再平衡，可以从波动中创造收益。这不是预测未来，而是应对当下。"
> — 基于 Claude Shannon 的投资思想

### 核心原理：波动不是风险，而是收益来源

大多数人认为波动是坏事——价格忽上忽下让人焦虑。但香农的恶魔揭示了一个反直觉的真相：**波动本身可以成为收益的来源**，前提是你有正确的应对机制——再平衡。

#### 波动拖累：为什么"涨10%再跌10%"不是0%

这是理解网格交易的第一道门槛。直觉上，涨10%再跌10%应该回到原点，但事实并非如此：

```
初始资金: ¥10,000
涨10% → ¥11,000
跌10% → ¥9,900  ← 损失了¥100！

验证: 1.1 × 0.9 = 0.99 ≠ 1.0
```

波动越大，拖累越重：

| 涨跌幅度 | 结果 | 拖累 |
|---------|------|------|
| ±10% | 0.99 | -1% |
| ±20% | 0.96 | -4% |
| ±30% | 0.91 | -9% |
| ±50%/±33.3% | 0.998 | ≈0%（但波动率高达42%） |

**关键公式**：

```
几何收益 ≈ 算术收益 - (波动率² / 2)
```

这个公式告诉我们：波动拖累是**指数级增长**的。波动率从10%提到20%，拖累从0.5%跳到2%；提到40%，拖累变成8%。这不是线性关系，而是平方关系。

#### 再平衡如何"变废为宝"

再平衡的操作很简单：**当股票占比偏离目标时，卖出过多的股票或买入过少的股票，恢复目标比例**。

看似平淡的操作，却产生了惊人的效果：

**无再平衡（全仓持有）**：
```
初始: ¥10,000 全仓股票
第1次: 涨50% → ¥15,000
第2次: 跌33.3% → ¥10,000  ← 回到原点，0收益
```

**50/50再平衡**：
```
初始: ¥5,000股票 + ¥5,000现金
涨50%: ¥7,500股票 + ¥5,000现金 = ¥12,500
再平衡: ¥6,250股票 + ¥6,250现金
跌33.3%: ¥4,167股票 + ¥6,250现金 = ¥10,417
再平衡: ¥5,208股票 + ¥5,208现金 = ¥10,417  ← 比原点多了¥417！

继续涨50%: ¥7,812股票 + ¥5,208现金 = ¥13,021 ← 正收益持续放大
```

**为什么？** 再平衡在涨的时候"锁定了一部分利润"（卖出），在跌的时候"摊低了成本"（买入）。每一次再平衡都在减少波动拖累。

数学上：算术收益减半（8.4%→4.2%），但波动率也减半（42%→21%），波动拖累从8.82%降到2.21%，净效果 = 4.2%-2.21% = **约2%的正收益**。

拖累减少了**75%**（8.82→2.21），而算术收益只减少了**50%**（8.4→4.2）。这就是再平衡的"不对称优势"。

#### 网格交易：再平衡的具体实现

网格交易是香农的恶魔在实践中最常见的应用形式：

```
核心规则：
1. 设定价格区间和网格密度
2. 价格触及下方网格线 → 买入（低买）
3. 价格触及上方网格线 → 卖出（高卖）
4. 每次交易后，仓位偏离目标 → 再平衡
5. 在震荡行情中反复收割波动
```

**等差网格 vs 等比网格**：

| 类型 | 间距公式 | 适用场景 | 优势 |
|------|---------|---------|------|
| 等差 | `(上限-下限)/(N-1)` | 低波动标的（国债、蓝筹） | 简单直观 |
| 等比 | `上限/下限^(1/(N-1))` | 高波动标的（加密、成长股） | 百分比对称，不会"破网" |

等比网格的关键优势：无论价格在哪个位置，相邻网格之间的**百分比间距是相同的**。这意味着¥5到¥5.5（10%）和¥10到¥11（10%）触发相同的交易信号——这在高波动标的中更加合理。

#### Parrondo悖论：两个输的策略如何组合成赢

Parrondo悖论与香农的恶魔有深层联系，也是本平台探索的方向：

```
Game A: 赢概率49% → 长期输钱
Game B: 资本%3==0时赢16%，否则赢75% → 长期也输钱
交替玩A+B → 反而可能赢钱！
```

映射到网格交易：

- **低波动网格盈利** ≈ Game A（高频小额赢）
- **高波动止损亏损** ≈ Game B（低频大额输）
- **跨资产再平衡** ≈ A+B组合（低波动资产稳定盈利 + 高波动资产间歇亏损 → 组合反而盈利）

**核心启示**：多资产网格策略的价值不在单一资产盈利，而在跨资产再平衡产生的Parrondo效应。相关性越低、波动差异越大的资产组合，再平衡效果越显著。

### 📐 核心计算公式

#### 1. 波动拖累（Volatility Drag）

```
g ≈ μ - σ²/2

g = 几何平均收益（实际复利收益）
μ = 算术平均收益
σ = 年化波动率
```

#### 2. 再平衡溢价（Rebalancing Premium）

```
RP ≈ (1/2) × σ₁ × σ₂ × (1 - ρ) × f

σ₁, σ₂ = 两资产波动率
ρ = 相关系数（越低溢价越高）
f = 再平衡频率
```

**直觉理解**：相关性低→再平衡收益高。两只股票同涨同跌（ρ=1），再平衡无效；一涨一跌（ρ=-1），再平衡效果最大。

#### 3. 凯利最优仓位（Kelly Criterion）

```
f = (bp - q) / b

f = 最优投资比例
b = 赔率（赢时净收益/本金）
p = 胜率
q = 1 - p
```

在网格交易中，凯利公式告诉我们：**仓位管理比择时更重要**。过度投入（f太大）会导致波动拖累吞噬收益，投入不足（f太小）则浪费了波动收割的机会。

#### 4. 最优再平衡阈值

```
阈值 ≈ √(交易成本 × 8 / σ²)

简化版: 当偏离 > 阈值时触发再平衡
阈值应覆盖交易成本并留有收益空间
```

#### 5. ATR止损止盈

```
止损价 = 入场价 - ATR × 止损乘数
止盈价 = 入场价 + ATR × 止盈乘数

ATR = Average True Range（平均真实波幅）
反映近期市场波动程度，动态调整止损距离
```

#### 6. 等比网格间距

```
网格线[i] = 下限 × (上限/下限)^(i/N)

i = 网格序号 (0, 1, ..., N)
N = 格数量
每格百分比间距 = (上限/下限)^(1/N) - 1
```

## 🚀 使用指南

### 安装

```bash
git clone https://github.com/reven-tang/gtap.git
cd gtap
python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

### 启动

```bash
streamlit run app.py
```

浏览器打开 `http://localhost:8501`

### 侧边栏配置

平台侧边栏分为三个按用户意图组织的区域：

#### 🎯 基础设置
- **股票代码**: 如 `sh.600958`（东方证券）
- **日期范围**: 回测起止日期
- **投入总资金**: 策略总预算
- **初次买入股数/每格交易股数**: 初始和每次交易的股数
- **数据源**: BaoStock(A股免费) / YFinance(港美股) / AkShare(A股增强)

#### 📊 交易策略
- **自动网格范围**: 开启后基于ATR自动计算上下限（推荐）
- **网格数量**: 网格密度，越多越频繁但单笔利润越小
- **网格间距**: 等差（适合蓝筹）或等比（适合高波动标的）
- **策略模式**:
  - `经典网格`: 传统低买高卖
  - `阈值再平衡`: 香农恶魔核心模式——偏离目标比例超过阈值时触发再平衡
  - `周期再平衡`: 定期再平衡（如每月）
- **目标股票比例**: 再平衡模式下的目标配置（如50%股票+50%现金）
- **再平衡阈值**: 偏离多少时触发（如5%）
- **仓位模式**: 固定股数/固定金额/比例仓位（再平衡模式推荐比例仓位）
- **ATR止损止盈**: 动态止损止盈保护

#### ⚙️ 高级设置（折叠）
- 交易费用（佣金/过户费/印花税）
- K线频率（5分钟~月线）
- 复权类型
- 季频财务数据

### 推荐配置示例

**蓝筹股震荡策略（东方证券）**：
```
策略模式: 阈值再平衡
目标比例: 50%
再平衡阈值: 5%
网格间距: 等差
数据源: BaoStock
```

**高波动标的策略（加密/成长股）**：
```
策略模式: 阈值再平衡
目标比例: 40%  ← 更保守，留更多现金缓冲
再平衡阈值: 8%  ← 波动大，阈值宽一些减少过度交易
网格间距: 等比  ← 百分比对称更合理
自动网格范围: 开启  ← 让ATR决定范围
```
