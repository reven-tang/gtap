# GTAP v0.2.0 架构重构 — 模块化设计

## 拆分策略

### 目标结构
```
gtap/
├── src/gtap/
│   ├── __init__.py         # 包入口，导出公共 API
│   ├── config.py           # 配置类（默认参数）
│   ├── data.py             # 数据获取模块（get_stock_data）
│   ├── fees.py             # 费用计算模块（calculate_fees）
│   ├── grid.py             # 网格回测引擎（grid_trading）
│   ├── metrics.py          # 绩效指标计算（Sharpe/MaxDD等）
│   ├── plot.py             # 图表绘制封装
│   └── exceptions.py       # 自定义异常
├── app.py                  # Streamlit UI（精简后的 main）
├── tests/                  # 单元测试（待创建）
├── requirements.txt
├── pyproject.toml
└── README.md
```

### 模块职责

| 模块 | 职责 | 输入 | 输出 | 依赖 |
|------|------|------|------|------|
| config | 配置类、默认参数、参数验证 | - | Config 类 | pydantic（可选，先原生） |
| data | baostock 数据获取、清洗、缓存 | code, start, end, freq | DataFrame + 元数据 | baostock, pandas |
| fees | 交易费用计算（佣金/过户费/印花税） | amount, is_buy, code, rates | fee breakdown | - |
| grid | 网格交易回测引擎 | data, params | 交易记录 + 资产曲线 | pandas, numpy |
| metrics | 绩效指标计算 | asset_values, trades, params | 指标字典 | pandas, numpy |
| plot | 图表绘制（K线/资产曲线/网格） | data, trades, metrics | Plotly 图表对象 | plotly, mplfinance |
| exceptions | 自定义异常类 | - | 异常类型 | - |
| app | Streamlit 界面逻辑 | user input → 调用各模块 | 渲染结果 | streamlit |

### 拆分顺序（依赖拓扑）
1. exceptions（零依赖）
2. config（零依赖）
3. fees（零依赖）
4. grid（依赖 data/fees）
5. metrics（依赖 grid）
6. data（独立，可单独测试）
7. plot（依赖 data/grid）
8. app（顶层组装）
